## Description

This repository is an implementation of a user micro-service using Nest.js and gRPC. I have used node v16.16.0 for this codebase

## Prerequisites

Node version > 16.x

## Installation

```bash
$ npm install
$ npm run proto:install
$ npm run proto:user
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
